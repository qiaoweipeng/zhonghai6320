# 消防报警控制器仿真系统

基于 Vue 3 + Tailwind CSS 构建的火灾报警控制器仿真界面。

## 技术栈

- **Vue 3** - 渐进式 JavaScript 框架（通过 CDN 引入）
- **Tailwind CSS** - 实用优先的 CSS 框架（通过 CDN 引入）
- **HTML5** - 超文本标记语言

## 运行方式

由于使用了 CDN 引入外部资源，需要通过 HTTP 服务器运行：

### 方法一：Python HTTP 服务器

```bash
cd /Users/jackqiaosan/Desktop/code/众海_6320_火灾报警控制器
python3 -m http.server 8000
```

然后在浏览器中访问 `http://localhost:8000/index.html`

### 方法二：VS Code Live Server

1. 安装 `Live Server` 插件
2. 右键点击 `index.html` → "Open with Live Server"

| 指示灯 | 20px | 10px | 含标签高度约 25px |

### 实际显示尺寸（占屏幕 90% 宽度）

实际显示尺寸根据屏幕宽度自适应计算：

**计算公式：**
- 显示宽度 = 屏幕宽度 × 90%
- 显示高度 = 显示宽度 × 0.4155（保持原始宽高比）

**不同屏幕分辨率下的实际尺寸：**

| 屏幕分辨率 | 显示宽度 | 显示高度 |
|------------|----------|----------|
| 1920 × 1080 | 1728px | 718px |
| 1366 × 768 | 1229px | 511px |
| 1440 × 900 | 1296px | 538px |
| 2560 × 1440 | 2304px | 957px |

### 组件尺寸（固定像素）

当前代码中使用的组件尺寸是固定像素值，不会随屏幕宽度自动缩放：

| 元素 | 宽度 | 高度 | 字体大小 |
|------|------|------|----------|
| 指示灯 | 40px | 20px | 17px |
| 大按钮 | 44px | 44px | 17px |
| 屏幕按钮 | 122px | 47px | 20px |
| 小按键 | 49px | 49px | 14px |

> **注意**：组件尺寸为固定像素值，不会随屏幕宽度变化而自动调整。如需实现完全响应式布局，需将组件尺寸改为 `vw` 单位。

### 缩放比例说明

项目使用 `vw`（视口宽度）作为主容器单位，主容器会随屏幕宽度自动调整：

- 主容器宽度：`90vw`（占屏幕宽度的 90%）
- 主容器高度：`calc(90vw * 0.415)`（保持原始宽高比 1745:725）
- 红色曲线：使用 SVG 实现，支持任意宽度缩放
- 内边距、间距等：部分使用 `vw` 单位，部分使用固定像素值

## 开发说明

### 修改尺寸

如需调整界面尺寸，修改 `js/app.js` 中以下参数：

1. **主容器尺寸**（第 133 行和第 228 行）：
   - `w-[90vw]` - 修改宽度百分比
   - `height: calc(90vw * 0.415)` - 修改高度计算比例

2. **组件尺寸**（第 3-85 行）：
   - `LightItem` - 指示灯组件
   - `BigBtn` - 大按钮组件
   - `ScreenBtn` - 屏幕按钮组件
   - `SmallBtn` - 小按键组件

### 添加新功能

1. 在 `js/app.js` 中添加新的组件或修改现有组件
2. 在 `AlarmPage` 或 `MenuPage` 中注册并使用新组件
3. 在 `setup()` 函数中添加状态和方法

## 注意事项

1. **浏览器兼容性**：建议使用现代浏览器（Chrome、Firefox、Edge）
2. **网络要求**：需要联网以加载 Vue 3 和 Tailwind CSS 的 CDN 资源
3. **本地运行**：必须通过 HTTP 服务器运行，不能直接双击打开 HTML 文件
4. **响应式**：界面会根据屏幕宽度自动调整大小，但保持原始宽高比
