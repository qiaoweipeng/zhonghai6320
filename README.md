# 消防报警控制器仿真系统

基于 Vue 3 + Tailwind CSS 构建的火灾报警控制器仿真界面。

## 技术栈

- **Vue 3** - 渐进式 JavaScript 框架（通过 CDN 引入）
- **Tailwind CSS** - 实用优先的 CSS 框架（通过 CDN 引入）
- **Web Audio API** - 按键音效合成

## 运行方式

由于使用了 CDN 引入外部资源，需要通过 HTTP 服务器运行：

### 方法一：Python HTTP 服务器

```bash
cd /Users/jackqiaosan/Desktop/code/众海_6320_火灾报警控制器
python3 -m http.server 8000
```

然后在浏览器中访问 `http://localhost:8000/index.html`

### 方法二：VS Code Live Server

1. 安装 `Live Server` 插件
2. 右键点击 `index.html` → "Open with Live Server"

## 界面尺寸

主容器采用固定像素尺寸 `1750 × 720`，内部组件均为固定像素值，不会随屏幕自适应缩放。

| 元素 | 宽度 | 高度 | 备注 |
|------|------|------|------|
| 主容器 | 1750px | 720px | 圆角25px，深灰背景 #b6b6b6 |
| 屏幕区域 | 65% | 满高 | 黑色边框，内部蓝色显示区 #c8e8ff |
| 右侧面板 | 35% | 满高 | 指示灯 + 小按钮 |
| 指示灯 | - | - | 5×3 网格，无名称的用 visibility:hidden 占位 |
| 小按钮 | 40px | 40px | 圆形，5×4 网格 |
| 屏幕按钮 | 90px | 47px | 蓝色圆角矩形，5个竖排 |

## 页面导航

```
警情页面 (alarm)
    │  点击屏幕第5个按钮
    ▼
菜单页面 (menu)
    │  白框选中"信息查询" + 按小按钮"确认"
    ▼
信息查询页面 (search)
```

- **警情 ↔ 菜单**：点击屏幕右侧第5个物理按钮切换
- **菜单 → 信息查询**：方向键移动白框到「信息查询」，按小按钮「确认」进入
- 白框默认停在第一个图标上，方向键（▲▼◀▶）移动选择

## 功能说明

### 小按钮（20个）

| 按钮 | 功能 |
|------|------|
| 复位 / 声光 / 自检 / 取消 | 功能键 |
| 确认 | 确认当前操作（菜单页选中信息查询时进入查询页） |
| 消音 | 切换消音指示灯状态 |
| 1 / 2ABC / 3DEF ... | 数字字母输入键 |
| ▲ ▼ ◀ ▶ | 方向键，移动菜单/查询页面的白框选择 |
| 0 | 数字键 |

### 屏幕按钮（5个，竖排）

第5个按钮用于警情页面与菜单页面之间的切换。

### 按键音效

- 所有按钮按下时播放 1400Hz 正弦波蜂鸣音，持续 0.1 秒
- 使用共享 `AudioContext`，兼容浏览器自动播放策略
- 首次点击时自动 `resume()` 激活音频上下文

### 指示灯状态

| 指示灯 | 颜色 | 状态说明 |
|--------|------|----------|
| 火警 | 红色 | 火灾报警激活 |
| 启动 | 红色 | 设备启动中 |
| 故障 | 黄色 | 系统故障 |
| 监管 | 黄色 | 监管模式 |
| 屏蔽 | 黄色 | 设备屏蔽 |
| 主电 | 绿色 | 主电源正常 |
| 反馈 | 红色 | 设备反馈 |
| 自动 | 红色/灰色 | 自动模式（红）/手动模式（灰） |
| 消音 | 红色/灰色 | 消音开启（红）/关闭（灰） |
| 备电 | 灰色 | 备用电源 |
| 自检 | 灰色 | 自检状态 |
| 系统故障 | 灰色 | 系统故障 |

## 开发说明

### 组件依赖顺序

`index.html` 必须按以下顺序加载脚本（基础组件在前，复合组件在后）：

1. `audio.js` - 音频工具（最先加载，供按钮组件使用）
2. `LightItem.js` / `SmallBtn.js` / `ScreenBtn.js` - 基础组件
3. `AlarmPage.js` / `MenuPage.js` / `SearchPage.js` - 页面组件
4. `Screen.js` - 屏幕组件（依赖上述页面）
5. `HomePage.js` - 首页（依赖 Screen）
6. `app.js` - 应用入口

### 添加新页面

1. 在 `js/pages/` 下新建组件文件，参考 `MenuPage.js` 结构
2. 在 `index.html` 中添加 `<script>` 引用
3. 在 `Screen.js` 的 `components` 中注册，并添加 `v-else-if` 分支
4. 在 `HomePage.js` 中添加对应的 `selectedIndex` 状态和导航逻辑

## 注意事项

1. **浏览器兼容性**：建议使用现代浏览器（Chrome、Firefox、Edge）
2. **网络要求**：需要联网以加载 Vue 3 和 Tailwind CSS 的 CDN 资源
3. **本地运行**：必须通过 HTTP 服务器运行，不能直接双击打开 HTML 文件
4. **音频策略**：浏览器要求用户交互后才能播放音频，首次点击按钮会激活音频上下文
5. **固定尺寸**：界面为 1750×720 固定像素，小屏设备需滚动查看
